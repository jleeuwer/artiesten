# ART-013B-1 Implementation Release Notes

Datum: 2026-07-12

Volledig handmatig musician-in-band relatiebeheer is geïntegreerd in de Artiesten-app. De release bevat migratie/preflight, modulaire CRUD API, band- en persoonsweergave, rollen/periodes/bronnen/notities, duplicate- en overlapdetectie, optimistic concurrency, tests en documentatie. Geen externe provider schrijft naar `musician_in_band`.
