# ART-UI-2 Fase 3/4 — Design release notes

Datum: 2026-07-14

Deze designrelease werkt de vervolgfase van ART-UI-2 uit. De artiestentabel krijgt in de codesprint een responsive begrensde viewport met sticky header en expliciete regels voor scrollpositiebehoud. Standalone gebruikt de documentscroll; Shellstarter embedded gebruikt één app-workspacescroll. Detailmodules blijven vrij van verticale scrollcontainers.

Er zijn 80 functionele en technische testcases toegevoegd. De sprint vereist geen database- of API-migratie.
